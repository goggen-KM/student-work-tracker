初期状態は「未入力」です。limits.csv を同じ場所に置くと、日別上限を自動で読み込みます。

初回登録

学生名・学籍番号・クラスは最初に1回だけ入力します。入力内容はこのスマホに自動保存されます。

アルバイト情報（最大3件）

アルバイト名と時給を入力します。時給は数字だけ入力してください。例：1100

一括設定

※「未入力だけをなし」は、すでに「あり」で決定した記録を消しません。

日別上限CSV

管理者が GitHub に limits.csv を置くと、アプリ起動時に自動で読み込みます。学生は操作不要です。
CSV形式：
date,limit
2026-07-01,28
2026-07-20,40
日別上限CSV：未読込

出力

PDFはブラウザの印刷画面で「PDFとして保存」を選んでください。

CSVデータ
